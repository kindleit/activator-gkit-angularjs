## How to run the code

1. Open a console & run:

```bash

~$ git clone https://github.com/kindleit/todo-gkit-angularjs.git
~$ cd todo-gkit-angularjs
~$ activator
~$ run

```
---

2. Open a console & run: 

```bash
~$ cd todo-gkit-angularjs/web-ui/
~$ npm install -g yo              
~$ npm install -g grunt-cli      
~$ npm install -g bower           
~$ npm update
~$ bower update
~$ grunt serve

```
